Terminal 用户必备

常用命令速查

macOS / Linux 重度终端用户？这里汇总了 OpenClaw CLI 最常用的命令，点击即可复制。

openclaw configure

打开龙虾配置向导

openclaw onboard

打开龙虾配置页面

openclaw dashboard

打开龙虾控制面板

openclaw config get

查看龙虾所有配置

openclaw config get --section XX

查看龙虾指定配置（例如 --section models）

openclaw config set XX

设置龙虾指定配置

openclaw config set models.default mistral:mixtral-8x7b

设置默认模型

openclaw config unset models.fast

删除特定配置项

openclaw config unset models

重置某个配置节点

openclaw doctor

对龙虾进行健康检查

openclaw --help

查看龙虾所有命令

openclaw gateway start

启动 Gateway（默认端口 18789）

openclaw gateway restart

网关重启，常用于让龙虾配置生效

openclaw logs --lines 50

查看最近 50 行日志，调试 Bug 必备

openclaw skills search XX

搜索龙虾技能

openclaw skills list

查看小龙虾安装的所有技能包

openclaw skills install XX

安装龙虾技能

openclaw skills uninstall XX

卸载龙虾技能

openclaw skills show XX

查看龙虾技能详情

openclaw skills update

更新所有已安装技能

openclaw sessions

查看龙虾所有的会话

openclaw message send --message "Hello"

发送消息到当前会话

openclaw channels show XX

展示龙虾特定频道

🔍 没有匹配的命令，试试其他关键词
