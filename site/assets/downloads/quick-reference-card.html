🚀 启动问题

  • 打不开/报错：检查是否完整解压到新目录，不要在压缩包内运行
  • SmartScreen 拦截：点击「更多信息」→「仍要运行」
  • Gatekeeper 拦截（macOS）：系统设置 → 隐私与安全性 → 仍要打开
  • 正确端口：http://127.0.0.1:18789（不是 3000）

🔑 API Key 问题

  • 国内模型：豆包/通义千问/DeepSeek，无需翻墙
  • Key 无效：检查是否复制完整，前后无空格
  • 测试失败：先确认网络能访问对应服务商
  • 获取 Key：去对应模型官网控制台创建 API Key

🧩 Skill 安装问题

  • 找不到 Skill：用 ClawHub 搜索，确认技能名拼写正确
  • 安装失败：检查网络连接，重试或换源
  • 装上后报错：看错误信息，通常是配置项缺失
  • 推荐先装：搜索、文件、Git、写作类 Skill

⚡ 常用命令

  • Windows 启动：双击 launch.bat
  • macOS 启动：右键打开 launch.command
  • 查看日志：运行 diagnose.bat（Windows）
  • 手动安装 Skill：openclaw skills install <技能名>
