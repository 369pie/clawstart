# ClawStart 首次启动检查清单

**版本**：v0.2.3  
**最后更新**：2026-03-15  
**适用平台**：Windows 10/11 64 位、macOS 12+

---

## 下载与解压

- [ ] 已从 GitHub Releases 下载最新一键包
- [ ] 已完整解压到**新的空目录**（不要在压缩包内直接运行）
- [ ] 解压路径不包含中文或特殊字符

---

## 首次启动

### Windows 用户
- [ ] 双击 `launch.bat` 启动
- [ ] 如遇 SmartScreen 拦截，点击「更多信息」→「仍要运行」
- [ ] 等待命令行窗口显示「OK OpenClaw 已启动」
- [ ] 浏览器自动打开 `http://127.0.0.1:18789`

### macOS 用户
- [ ] 右键点击 `launch.command` → 打开
- [ ] 如遇 Gatekeeper 拦截，前往「系统设置」→「隐私与安全性」→「仍要打开」
- [ ] 等待终端窗口显示「OK OpenClaw 已启动」
- [ ] 浏览器自动打开 `http://127.0.0.1:18789`

---

## 模型配置

- [ ] 已选择模型服务商（豆包/通义千问/DeepSeek 等）
- [ ] 已填入正确的 API Key
- [ ] 已点击「测试连通性」并看到成功提示
- [ ] 如测试失败，已查看排障页对应解决方案

---

## 第一次对话

- [ ] 在对话界面输入一条测试消息（如「你好，介绍一下你自己」）
- [ ] 已收到正常回复（不是报错）
- [ ] 回复内容合理，不是乱码或空白

---

## 完成标准

**✅ 装好了** = 能拿到第一条可用结果，不是「能打开界面」

如果以上所有勾选都完成，恭喜你已经成功安装并跑通 ClawStart！
