Windows 一键包步骤

  1. 下载安装包

    打开最新 Release，选择 Windows 对应的 `ClawStart-Windows-*.zip` 下载。

  2. 解压并运行启动器

    将 zip 解压到任意目录，双击 launch.bat。如果 SmartScreen 拦截，点击「更多信息」→「仍要运行」。

  3. 完成首次引导

    按引导选择模型服务商、填入 API Key、测试连通性。成功后浏览器会自动打开。

命令行安装步骤

  1. 安装 Node.js

    如果还没有 Node.js，去 nodejs.org 下载 LTS 版本安装。

    或者用包管理器安装：

    # Winget (推荐) winget install OpenJS.NodeJS.LTS # Scoop scoop install nodejs-lts # Chocolatey choco install nodejs-lts

  2. 全局安装 OpenClaw

    npm install -g @anthropic/openclaw

  3. 启动服务

    openclaw start

    浏览器会自动打开配置界面。

常见问题

Windows SmartScreen 阻止安装？

弹出"Windows 已保护你的电脑"时，点击「更多信息」→「仍要运行」即可。底层的 OpenClaw 是开源软件，源码可在 GitHub 查看。

安装后运行报错 "找不到 openclaw 命令"？

关闭并重新打开命令提示符/ PowerShell，让 PATH 环境变量生效。如果仍然找不到：

# 检查 npm 全局路径 npm config get prefix # 将该路径添加到 PATH 环境变量 setx PATH "%PATH%;%APPDATA%\npm"
杀毒软件误报病毒？

部分杀毒软件可能会误报。你可以：

  • 将当前启动器文件夹添加到信任列表/白名单
  • 或者使用命令行版本（npm 安装）
  • 从 GitHub 下载源码自行编译
