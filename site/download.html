下载后如果卡住，先按这个顺序分流

1. 先确认是不是下错平台

Windows 和 macOS 用户优先走一键包；Linux 用户优先走 Beta 安装脚本，不建议纯小白直接上手动 CLI 路径。

2. 启动失败先看权限和杀软

Windows 先看 SmartScreen，macOS 先看 Gatekeeper；别先删目录，先排权限。

3. 连接失败先看模型和网络

API Key、代理、证书、国内网络是高频问题，排障页里已经有对应规则。

按问题去排障 → 回到快速开始 →

系统要求

下载前先看这几项

系统 Windows 10/11、macOS 12+，Linux 当前主推荐路径为 Beta 安装脚本
内存 ≥ 4GB RAM
磁盘 ≥ 500MB 可用空间
网络 如果使用在线模型，需要能访问对应 API；国内用户优先选择国内模型提供商

下载后这样做

不要下载完就停住

Step 1

解压或打开安装包

Windows 推荐解压后双击 `launch.bat`，macOS 推荐右键打开 `launch.command`。

Step 2

按引导配置模型

选择模型服务商，填入 API Key，先确保能完成第一次连接。

Step 3

卡住就去排障

不要反复乱试，直接按排障页现象分类排查，效率更高。

查看快速开始 → 遇到问题？去排障 →

装好后先别急着折腾高级功能

先按快速开始跑通第一轮，再去教程页做第一件有成就感的事。
