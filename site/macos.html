macOS 一键包步骤

  1. 下载安装包

    打开最新版本页面，选择适合自己芯片架构的 macOS 压缩包下载。第一次安装优先走一键包，最简单省心。

  2. 解压后右键打开启动器

    先解压下载的压缩包，再右键打开启动脚本（launch.command）。首次若被拦截，请在系统设置里选择"仍要打开"。出现拦截是正常的，不代表安装包有问题。

  3. 完成首次引导并拿到第一次正常回复

    按引导选择模型服务商、填入 API Key（服务商给你的专属密码）、测试连通性。浏览器打开后，先确认你已经拿到一条正常回复，这才算真的跑通。

高级：命令行安装步骤

💡 适合谁 以下步骤面向熟悉终端和 Homebrew 的开发者。如果你是普通用户，请直接使用上面的一键包。

  1. 安装 Node.js

    OpenClaw 需要 Node.js 22 或更高版本。推荐用 Homebrew 安装：

    brew install node

    没有 Homebrew？去 nodejs.org 下载 pkg 安装包。

  2. 全局安装 OpenClaw

    npm install -g openclaw@latest

    也可以用 pnpm：pnpm add -g openclaw@latest

  3. 运行引导向导

    openclaw onboard --install-daemon

    向导会引导你完成 Gateway、Workspace、频道和 Skill 的配置。

常见问题

提示"无法打开，因为无法验证开发者"

两种解决方式：

方式一：打开「系统设置 → 隐私与安全性」，往下翻找到提示，点击「仍要打开」。

方式二：如果你熟悉终端操作，也可以在终端里对解压目录执行一条清除安全标记的命令后再重试。

Apple Silicon 和 Intel 有什么区别？

当前一键包按芯片架构分别提供下载。大多数用户只要按下载页的设备类型选择对应版本即可，不需要理解太多技术差异。

安装后找不到 openclaw 命令？

如果你走的是一键包，其实不需要手动使用命令行；直接运行启动脚本（launch.command）就够了。只有命令行安装的用户，才需要排查环境变量设置。
