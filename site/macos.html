macOS 一键包步骤

  1. 下载安装包

    打开最新 Release，选择适合自己芯片架构的 macOS `tar.gz` 产物下载。

  2. 解压并右键打开

    解压下载的压缩包，右键打开 launch.command。首次若被拦截，请在系统设置里选择“仍要打开”。

  3. 完成首次引导

    按引导选择模型服务商、填入 API Key、测试连通性。成功后浏览器会自动打开。

命令行安装步骤

  1. 安装 Node.js

    如果还没有 Node.js，先安装 LTS 版本：

    brew install node

    没有 Homebrew？去 nodejs.org 下载 pkg 安装包。

  2. 安装底层 OpenClaw CLI

    npm install -g @anthropic/openclaw

  3. 启动服务

    openclaw start

    浏览器会自动打开配置界面。

常见问题

提示"无法打开，因为无法验证开发者"

两种解决方式：

方式一：打开「系统设置 → 隐私与安全性」，往下翻找到提示，点击「仍要打开」。

方式二：如果你明确知道当前一键包解压目录，也可以在终端里对该目录执行 `xattr -cr` 后再重试。

Apple Silicon 和 Intel 有什么区别？

当前一键包按芯片架构分别提供下载。大多数用户只要按下载页的设备类型选择对应版本即可，不需要理解太多底层差异。

安装后找不到 openclaw 命令？

如果你走的是一键包路径，其实不需要手动使用 `openclaw` 命令；直接运行 `launch.command` 就够了。只有命令行安装用户，才需要继续排查 PATH 设置。
