# Code of Conduct

## Our goal

ClawStart is built to make installation and onboarding more accessible for ordinary users and beginners.

We want the project community to feel:

- welcoming
- practical
- respectful
- focused on helping people succeed

## Expected behavior

Examples of good community behavior:

- being respectful and constructive
- focusing on the user problem, not the person
- giving actionable feedback
- helping newcomers understand context
- keeping discussions aligned with the project scope

## Unacceptable behavior

Examples of unacceptable behavior:

- harassment or personal attacks
- insulting or demeaning language
- trolling, deliberate provocation, or bad-faith derailment
- publishing private information without permission
- repeatedly pushing unrelated agendas into project discussions

## Scope

This Code of Conduct applies to:

- issues
- pull requests
- discussions
- review comments
- other public community spaces related to this repository

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, commits, code, or other contributions that do not align with this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainers through the repository issue tracker first, or through a private maintainer contact channel once it is published.

## Notes

This project prefers calm, high-signal collaboration.

If a discussion stops helping users, it is probably moving in the wrong direction.
