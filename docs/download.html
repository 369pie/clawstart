下载后遇到问题？按这个顺序排查

1. 先确认是不是下错平台

Windows 和 macOS 用户优先走一键包；Linux 用户可通过安装脚本配置。纯新手建议先从一键包开始，更省心。

2. 启动被拦截？看系统安全设置

Windows 上是 SmartScreen（系统自带的安全拦截），macOS 上是 Gatekeeper（系统自带的安全验证）。按提示点几下就能通过。

3. 连接失败？检查模型密钥和网络

API Key（模型服务商给你的密码）、网络连通性是高频问题，排障页有详细检查清单。
