第一次任务模板

如果你装好以后不知道先问什么，就别空想。直接挑一个和你最近最常做的事最接近的模板照抄。

使用方法 先选一个离你最近的场景，直接复制一条模板去问。第一次只追求“我真的得到一个像样结果”，不要一上来追求复杂工作流。

先做选择

按你最近最常做的事来挑

写作 / 内容

适合创作者、运营、学生和日常要写文案的人。

办公 / 整理

适合会议纪要、提炼重点、输出待办这类场景。

开发 / 技术

适合需要解释报错、看仓库、写小脚本的用户。

进阶入口

成功标准

第一次任务做到这 3 件事就够了

1

你拿到了一份能读、能判断、像样的结果

2

你知道这次结果哪里好、哪里还不够

3

你已经知道下一步是换模板、装 Skill，还是去排障

什么时候别继续试模板 如果你连续两三次都拿不到正常回复，或者模型连接明显不稳定，就先去 排障页，不要把问题误以为是模板不会写。
