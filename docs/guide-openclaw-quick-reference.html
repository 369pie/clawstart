OpenClaw 小白排障速查

来源：useclawstart.com/guide-openclaw-quick-reference.html

适合谁 刚装好就卡住的小白用户
使用方式 先按现象判断，再回对应教程
版本 v0.1 Quick Reference
先判断

是打不开、配不好，还是 Skill 装不上。

先走最短动作

小白优先回一键包路径，不要先折腾命令行。

先验证结果

每修一步就重新打开一次，不要堆一串改动。

打不开 / 启动失败

  • macOS 看系统设置里的“仍要打开”，或执行 `xattr -cr "/你的解压目录"`。
  • Windows 被 SmartScreen 拦截时，点“更多信息”再继续运行。
  • 如果反复被拦截，优先换全新目录重新解压。

模型或 API 连不上

  • 先检查 API Key 是否复制正确、额度是否正常。
  • 中国大陆用户优先用稳定在线模型，不要一开始就上本地模型。
  • 先用最简单请求验证连通，再回到复杂配置。

Skill 装不上

  • 第一次先装一个 Skill，不要一次装很多。
  • 优先选“适合小白 + 无需 API Key”的 Skills。
  • 装好后立刻测试，不通过就回到 Skills 指南排查。

命令行报错

  • 看到 `git@github.com`、`exit 128`，先把 Git 拉取方式切到 HTTPS。
  • 看到 `ENOENT`、`cannot find module`，优先重新下载一键包或重装底层 CLI。
  • 看到 `EPERM`，先关掉旧进程、杀软和占用目录。
