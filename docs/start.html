10 分钟快速开始

按这条最短路径把 ClawStart 跑起来，再完成第一次模型连接

💡 前置条件 你已经在 下载页 选好平台，并拿到了对应的一键包。

先按这个顺序检测

如果中途卡住，不要跳步。先确认当前停在哪一环，再决定是不是去排障页。

1

一键包是否下载正确，并且已经解压到新的目录

2

启动器是否能正常打开，没有被系统权限或杀软拦截

3

模型连接测试是否通过，API Key 和网络是否都正常

4

是否已经完成第一次任务，拿到第一条有结果的输出

💡 什么时候去排障 只有当你明确卡在上面某一步时，再去 排障页 按错误类型处理，这样会快很多。

配置引导

  1. 第一次启动

    Windows 用户双击 launch.bat，macOS 用户右键打开 launch.command。

    第一次启动会自动进入配置引导，不需要你手动编辑配置文件。

    如果这一步就打不开，先去看 排障页 里的“打不开 / 启动失败”。

  2. 选择 AI 模型供应商

    优先选择你当前最容易拿到 API Key 的服务商。中国大陆用户推荐先从国内模型提供商开始：

    • 🇨🇳 豆包 / 硅基流动 / DeepSeek / 自定义兼容接口
    • 🧠 已有海外 API 的用户也可以继续使用现有服务商
    • 🏠 想少花钱也可以后续切到本地模型

  3. 输入 API Key

    填入对应服务商的 API Key。密钥仅保存在本地，不会上传到任何第三方。

    ⚠️ 安全提示 不要将 API Key 分享给他人或提交到公开仓库。

    如果这里报错，优先检查是不是复制了前后空格、Key 已过期，或者当前网络无法访问对应接口。

  4. 测试连通性

    点击「测试连接」，确认 API Key 有效、网络可用、模型能正常返回结果。

    ✓

    测试成功：说明这台机器已经具备第一次可用条件

  5. 完成初始配置

    设置工作目录，选择是否开机自启，点击「保存并进入」。

第一个任务

配置完成后，试试这些：

# 让 AI 帮你总结一个网页 总结这篇文章的核心观点：https://docs.openclaw.ai # 或者让它帮你写一份 README 帮我写一份 Node.js 项目的 README 模板
💡 小技巧 按下 Ctrl + Space（Mac 是 Cmd + Space）可以快速唤醒全局输入框。

验证安装成功

完成以下 3 项，说明你已经跑通了：

1

应用正常启动，没有报错

2

模型连接测试通过，可以正常对话

3

完成了第一个任务，得到预期输出
