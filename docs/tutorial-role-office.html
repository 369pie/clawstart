办公用户路径

如果你是办公用户，先从这些任务开始

不要一开始就追求复杂自动化。先让 OpenClaw 帮你处理会议纪要、文档总结和待办提炼，再决定后面如何扩展。

最适合谁 适合办公室协作、项目跟进、会议纪要、行政支持、项目助理和需要反复整理资料的人。

先做什么

办公用户第一次最推荐的 3 类任务

🗒️

会议纪要转待办

把会议记录直接整理成“结论、待办、负责人、截止时间”的结构化结果。

📄

长文档转摘要

把长方案、汇报材料、政策文件和网页压成可直接转发和决策的简版摘要。

✅

碎片信息转行动清单

把零散聊天记录、邮件要点和会议讨论整理成下一步行动列表。

推荐 Skill

办公用户第一批最值得装的能力

Summarize

办公与内容处理

适合网页、PDF、图片、音视频和文档的快速总结，是办公用户最容易立刻感受到价值的 Skill 之一。

npx clawhub@latest install summarize
查看原始页面 →

Notion

协作与沉淀

适合把纪要、清单、项目资料和知识库写回团队协作系统，而不是停留在临时对话里。

npx clawhub@latest install notion
查看原始页面 →
