## 4. First GitHub release body

```md
## ClawStart v0.1.0

This is the first public release of ClawStart.

ClawStart is a Chinese-friendly installation entry and onboarding project for OpenClaw, focused on helping ordinary users and beginners get through the first setup path more reliably.

### What is included

- multi-page onboarding and troubleshooting site
- Windows packaging flow
- macOS packaging flow
- Linux Beta installer path
- installer validation workflows

### Recommended user path

1. open the site
2. choose the correct platform path
3. download the package or run the Linux Beta installer script
4. follow the quick start flow
5. if setup fails, go to troubleshooting

### Downloads

| Platform | File |
|---|---|
| Windows | `ClawStart-Windows-latest.zip` |
| macOS (Apple Silicon) | `ClawStart-macOS-Apple-Silicon-latest.tar.gz` |
| macOS (Intel) | `ClawStart-macOS-Intel-latest.tar.gz` |

### Linux Beta

Linux is currently a secondary path compared with Windows and macOS.

Recommended Linux flow:

1. install Node.js 20+
2. run the Linux Beta installer script
3. start from `~/.clawstart-linux-beta/launch.sh`

### Main pages

- Home
- Download
- Quick Start
- Troubleshooting
- Tutorial
- Resources

### Notes

- this is an early public release
- the project is currently optimized around first-run success, not feature completeness
- Windows and macOS are the primary paths
- Linux is available as a Beta path
```
